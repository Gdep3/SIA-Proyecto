## Gestión de inventario de un Supermercado

Pasos para poder ejecutar el proyecto:

1) Descargar el archivo zip.
2) Extraer el archivo en la carpeta que deseas.
3) Abrir tu IDE de preferencia, nosotros hicimos el proyecto en NetBeans 22 por lo que recomendamos ese.
4) Abrir el proyecto o importarlo desde el lugar donde dejaste el archivo zip.
5) Si todo salio bien, al apretar Run debería de iniciar el programa correctamente.


Descripción de los paquetes:
````
En el paquete Default, se encuentra la clase SiaProyecto, ahí se encuentra nuestro main.
En el paquete Clases, se encuentran las clases que utilizamos.
En el paquete Controlador, se encuentra el controlador de las ventanas.
En el paquete Excepciones, se encuentran todas las excepciones que se utilizaron.
En el paquete Ventanas, se encuentran los diseños de todas las ventanas.
````

*Programa fue testeado y ejecutado con "Netbeans IDE 22"*
